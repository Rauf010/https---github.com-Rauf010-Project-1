Interview Azfar

Project Geralateerd
Vraag 1: Hoe beviel de samenwerking?
Antwoord: De samenwerking verliep soepel, vooral omdat wij veel gecommuniceerd hebben als wij iets wilden aanpassen of bijvoegen.
Vraag 2: Waren er moeilijkheden tijdens het project?
Antwoord: In de html code was er niet echt een probleem, het was de css code wat ons liet puzzelen.
Vraag 3: Zijn er verbeterpunten voor in de toekomst?
Antwoord: Ja, pas toe wat je hebt geleerd wanneer het nog vers in je hoofd is. Zo onthoudt je het beter.
Vraag 4: Heeft dit project je geholpen met je ontwikkeling in programmeren?
Antwoord: Zeker! Tijdens het project heb ik ervaren welke moeite er nodig is om een website te programmeren.
Vraag 5: Was dit een leuke ervaring?
Antwoord: Ik vond het een leuke ervaring, maar ik had liever gewild dat je meer zelf een idee voor de website mocht maken.
Maar ik begrijp dat dit is voor onze ontwikkeling en om onze vaardigheden te testen.
Persoonlijke Vragen
Vraag 6: Wat doe je graag in je vrije tijd?
Antwoord: Voetballen, zwemmen, fietsen, gamen of met vrienden of familie zijn.
Vraag 7: Vind je deze opleiding (nog) leuk?
Antwoord: Ja, ik had al een idee om deze opleiding te volgen sinds ik zelf een beetje begon te programmeren op de middelbare school.
Vraag 8: Wat is je favoriete eten?
Antwoord: Mijn favoriete eten is Biryani, ik zou dit wel elke dag kunnen eten.
Vraag 9: Wat staat er op jou bucketlist?
Antwoord: Op hadj gaan, dit is in het samengevat een pelgrimstocht naar Mekka wat verplicht is op elke moslim.
Vraag 10: Wat irriteert jou?
Antwoord: Kleine jongens met een grote mond.
