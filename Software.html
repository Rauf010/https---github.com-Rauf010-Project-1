Uitleg gebruikte software

Office 365

Hier op het Techniek College Rotterdam maken wij gebruik van software. Dat zal nodig zijn aangezien wij de opleiding Software Developer volgen.
En juist daarom maken wij gebruik van de volgende software, die in geheel gratis zijn om te gebruiken en ideaal zijn voor opleidingen als deze.

Office 365 is een verzamel-app van meerdere software van Microsoft. Dankzij het Techniek College Rotterdam kunnen wij als zowel leerling als docent
gebruik maken van Office 365. Office 365 wordt niet alleen gebruikt voor school, het wordt wereldwijd gebruikt, ook bij bedrijven. Wij maken gebruik van Office 365
wanneer het aankomt op opdrachten, projecten of communicatie. De software in Office 365 zijn als volgt:

  • Microsoft Word,
  • Microsoft Powerpoint,
  • Microsoft Excel,
  • Microsoft Outlook,
  • Microsoft OneNote,
  • Microsoft Teams,

Virtualisatie

Zoals uitgelegd maken wij op het Techniek College Rotterdam veel gebruik van online software. Daarom wilt het TCR al het papierwerk,
denk maar aan huiswerk en absentie etc. allemaal online doen (virtualisatie). Zo gaat alles sneller en efficienter en ben je niet afhankelijk van een bepaalde locatie.
Als voorbeeld is er online les, i.p.v de klaslokalen kun je nu de les volgen op elke plek mogelijk. Als tweede voorbeeld, stel je hebt een vraag de leraar, i.p.v
de leraar te zoeken kun je heel simpel een e-mail sturen.

Alle software gratis

Het is u net al vermeld, de software is gratis. Dit heeft uiteraard geen invloed op de kwaliteit. De software zijn professioneel en makkelijk te gebruiken.
Zouden wij moeten betalen voor de software, was het een moeilijke opleiding voor velen. Daarom is dit een groot pluspunt voor elke opleiding op het Techniek College.

Visual Studio Code

Als Software Developer coderen wij, Visual Studio Code is een applicatie die dit mogelijk maakt. Wat VSCode speciaal maakt is dat wij alles kunnen back-uppen.
Dus mocht er iets gebeuren waardoor we onze bestanden kwijt zijn, kunnen wij onze projecten nog altijd terugvinden. Deze back-up wordt gemaakt in GitHub.
In GitHub staat alles zo geregeld dat je kunt zien waar welk bestand is in diverse mappen. Het is te vergelijken met je bestanden-app, alleen staan er in GitHub kopietjes.
